1. Informations sur le Réalisateur (Profil Participant)
Cette section identifie l'interlocuteur principal et garantit l'éligibilité juridique.
● Identité : Civilité (M/Mme/Iel), Prénom, Nom*.
● Validation d’Âge : Date de naissance* (Contrôle système : Minimum 18 ans
révolus).
● Contact : Email*, Téléphone*, Mobile*.
● Localisation : Adresse postale complète (Rue, Code Postal, Ville, Pays)*.
● Profil Professionnel : Métier actuel*.
● E-réputation & Réseaux : URLs YouTube, Instagram, LinkedIn, Facebook, X
(Optionnel).
● Suivi Marketing : "Comment avez-vous connu marsAI ?" (Menu déroulant
extensible)*.
● Engagement : Case à cocher pour l'adhésion à la Newsletter.


2. Métadonnées du Film
Ces champs constituent la fiche technique de l'œuvre pour le catalogue du festival.
● Titres : Titre original* et Traduction anglaise*.
● Spécifications Temporelles : Durée exacte* (Validation système : ≤ 60 secondes).
● Linguistique : Langue parlée/principale du film*.
● Thématique : Tags sémantiques relatifs au thème "Futurs souhaitables"*.
● Argumentaire (Pitch) :
○ Synopsis langue originale* (Max 300 caractères).
○ Synopsis anglais* (Max 300 caractères).

3. Déclaration d'Usage de l'IA (Module Spécifique)
Section fondamentale pour la transparence et l'analyse technique des œuvres.
● Classification de l'Œuvre :* Choix exclusif entre :
1. Génération intégrale (100% IA).
2. Production hybride (Prises de vues réelles + apports IA).
● Stack Technologique :* Liste détaillée des outils utilisés (Script, Image, Animation,
Audio). Texte libre (500 car. max).
● Méthodologie Créative :* Note de production décrivant l'interaction entre l'humain et
la machine. Texte libre (500 car. max).

4. Livrables Multimédias & Accessibilité
● Support Vidéo :* URL source YouTube (Format : Public ou Non-répertorié).
● Accessibilité (Sous-titrage) :
○ Case à cocher : "Présence de voix ou de textes nécessitant des sous-titres".
○ Module d'upload : Fichier source au format .srt.
● Identité Visuelle :
○ Vignette Officielle :* Upload d'image haute résolution (Format 16:9).
○ Galerie Médias : Possibilité d'importer jusqu'à 3 captures d'écran du film
(Stills).

5. Composition de l'Équipe (Crédits)
Permet de générer le générique et les invitations pour la remise de prix.
● Gestion des Collaborateurs : Module dynamique permettant d'ajouter des fonctions
(Scénariste, Monteur, Sound Designer, etc.).
● Données par collaborateur : Civilité, Prénom, Nom, Profession, Email.


Note à l'équipe technique : La validation du formulaire ne peut être effective que si
l'ensemble des champs marqués (*) sont renseignés et si le fichier .srt est fourni dès lors
que la case "Sous-titres" est cochée.
